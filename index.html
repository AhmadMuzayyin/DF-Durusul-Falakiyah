Kebijakan Privasi Durusul Falakiyah

1. Pendahuluan

Terima kasih telah menggunakan Durusul Falakiyah. Kami berkomitmen untuk melindungi privasi dan data pribadi Anda. Kebijakan privasi ini menjelaskan bagaimana kami mengumpulkan, menggunakan, dan melindungi informasi Anda.

2. Informasi yang Kami Kumpulkan

2.1 Data Lokasi

  • Kami mengumpulkan data lokasi geografis Anda untuk:
    • Menghitung waktu shalat yang akurat
    • Menyesuaikan jadwal berdasarkan zona waktu Anda
  • Data lokasi disimpan secara lokal di perangkat Anda
  • Anda dapat menonaktifkan izin lokasi kapan pun

2.2 Izin Perangkat

  • Lokasi: Untuk perhitungan waktu shalat
  • Notifikasi: Untuk mengirim pengingat waktu shalat
  • Overlay: Untuk menampilkan pengingat di atas aplikasi lain
  • Optimasi Baterai: Untuk menjaga keandalan notifikasi

3. Penggunaan Data

  • Data lokasi hanya digunakan untuk:
    • Perhitungan waktu shalat
    • Penyesuaian zona waktu
  • Tidak ada data yang dikirim ke server eksternal
  • Semua perhitungan dilakukan secara lokal di perangkat Anda

4. Penyimpanan Data

  • Data lokasi disimpan dalam database lokal
  • Data lama secara otomatis dihapus
  • Tidak ada penyimpanan data di cloud

5. Keamanan Data

  • Semua data disimpan dengan enkripsi lokal
  • Akses data dibatasi hanya untuk fungsionalitas aplikasi
  • Kami tidak menjual atau membagikan data Anda

6. Hak Pengguna

  • Anda dapat mencabut izin kapan pun
  • Fitur aplikasi akan disesuaikan jika izin dicabut
  • Anda dapat menghapus data lokasi melalui pengaturan aplikasi

7. Anak-Anak di Bawah 13 Tahun

  • Aplikasi tidak ditargetkan untuk anak-anak di bawah 13 tahun
  • Orang tua/wali disarankan mendampingi anak menggunakan aplikasi

8. Perubahan Kebijakan

  • Kami dapat memperbarui kebijakan privasi sewaktu-waktu
  • Perubahan akan diumumkan melalui pembaruan aplikasi
